## ✏프로젝트 소개

Web3 쇼핑몰은 Metamask 암호화폐 지갑을 사용하는 쇼핑몰 플랫폼입니다.
사용자는 웹3 지갑을 통해 웹에 로그인하고 상품을 구매 및 판매할 수 있습니다.
